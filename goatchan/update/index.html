Sistema de lenguajes

Queremos hacer de Goat-Chan un sistema de entretenimiento accesible para todo el mundo. Por ello, hemos decidido implementar un sistema de lenguajes a nuestra cibercabra favorita para que así pueda llegar a más usuarios.
La actualización traerá consigo los idiomas Español, Gallego e Inglés (US), y aceptaremos traducciones de la comunidad que se implementarán en futuras actualizaciones!

Estadísticas

Sí, damas y caballeros, ahora Goat-Chan guardará estadísticas de uso para diferentes fines, como Rankings o logros.
*Goat-Chan no guardará más información personal de la que le quieras dar

Sistema de logros

Los logros son una forma de recompensar la actividad y el uso de nuestra querida Goat-Chan. Habrá dos tipos diferentes de logros: Progresivos y de Obtención única

Los logros Progresivos son aquellos que mejoran su nivel con el tiempo. Estos logros pueden tener entre 3 y 5 niveles, los cuales estarán representados por diferentes insignias.
Los logros de Obtención única, como su nombre indica, son logros que sólo se podrán obtener una vez.

Suscripción Premium

El fin del programa Partner ha llegado, y con él ha llegado una nueva suscripción de pago: Goat-Chan Premium.
Esta suscripción traerá los beneficios que el programa Partner otorgaba, además de un logro y un hueco importante en nuestro corazón.

Personalización de perfil de usuario

Con la nueva actualización podrás personalizar aspectos de tu perfil como tu apodo, color favorito, fecha de cumpleaños... para así desbloquear ciertas funciones especiales dentro del bot.

Sistema de Roleplay 2.0

Para la 1.6 habremos reformado el sistema de Roleplay de Goat-Chan, con tal de crear una gran experiencia para nuestros usuarios.
El nuevo sistema tendrá una economía interna que relacionará los comandos de interacción con el propio sistema de economía. Podrás usar comandos de interacción para ganar dinero (o perderlo) y usar ese dinero para comprar objetos o tickets para obtener premios de temporada!
